# Python Libraries and DSA

A polished Python learning repository focused on data structures, algorithms, and practical library usage.

This workspace brings together hands-on practice files for core Python concepts, classic DSA topics, and real-world library exploration. It is designed as a personal study archive and a reference for building stronger problem-solving and coding skills.

## 🚀 What’s Included

### 1. Data Structures and Python Basics
Explore foundational Python concepts such as:
- Lists, tuples, sets, and dictionaries
- Strings and string methods
- Variables and operators
- Functions and assignments
- Exception handling
- Special operators and control-flow fundamentals

These examples live in the [python/datastructureslearning](python/datastructureslearning) folder.

### 2. LeetCode Practice
A collection of Python solutions for common coding interview problems, including:
- Two Sum
- Plus One
- Valid Palindrome
- Valid Anagram
- Longest Palindromic Substring
- Word Search
- And several other practice problems

These files are organized in the [python/leetcode problems](python/leetcode%20problems) folder.

### 3. Python Library Practice
Hands-on experiments with popular Python libraries and concepts such as:
- Object-oriented programming
- Collections framework
- Datetime utilities
- NumPy
- Pandas
- Pydantic
- Typing and annotations

These practice files are stored in the [python/projectlibpractice](python/projectlibpractice) folder.

## 🛠️ How to Use

1. Clone the repository to your local machine.
2. Open the project in VS Code or any Python IDE.
3. Run any Python file directly:

```bash
python "python/leetcode problems/plusone_66.py"
```

4. Explore the folders by topic and build your understanding step by step.
